    mapCsv

    • mapCsv(csvText?: string): false | MappedOutput

      Maps CSV text (or currently loaded CSV) using the current configuration and returns the result.

      Parameters

      • OptionalcsvText: string

        Optional CSV text to parse and map. If not provided, uses currently loaded CSV.

      Returns false | MappedOutput

      MappedOutput object with data, csv string, and validation info, or false if error or no csv loaded.

      • false
      • MappedOutput
        • csv: string
        • data: Csv
        • validation: ValidationResult

      Example

      // Map currently loaded CSV
      const result = mapper.mapCsv();
      if (result) document.getElementById('csv').value = result.csv;

      // Map new CSV text
      const csvText = 'Name,Email\nAlice,Alice@example.com';
      const result2 = mapper.mapCsv(csvText);
      if (result2) document.getElementById('csv').value = result2.csv;
