CSV Mapper Server Demo

This demo shows how CSV Mapper works with server uploads. Upload a CSV file, map the columns using the UI, and see the server response.

Upload CSV File

Expected Columns

This demo expects the following columns:

  • product_id - Unique product identifier
  • name - Product name
  • price - Product price
  • category - Product category
  • stock - Stock quantity

Try uploading a CSV with different column names to see the mapping interface in action!
